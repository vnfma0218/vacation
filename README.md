## 호텔 예약 어플리케이션

호텔을 예약할 수 있는 어플리케이션

## 소개

sanity io를 사용하여 호텔을 등록하였고 Next.js로 개발하여 해당 호텔 데이터를 보여주고 검색이 가능

## 개발기간

- 23.12.01 ~ 24.01.05

## 주요기능

- 지역별 호텔 검색
- 검색결과 내 필터링 적용(가격, 평점순 등)
- 호텔 상세 페이지
- 호텔 예약 페이지 및 호텔 예약 확인 페이지
